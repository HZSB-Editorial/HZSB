- title: 关于衡中
  details: 衡水中学是河北省一所知名高中，广义的衡中指河北衡水中学（公办高中）、衡水第一中学（民办高中）、衡水中学实验学校（民办初高中）、 衡水滏阳中学（民办高中），四个校区；狭义的衡中仅指河北衡水中学。
- title: 四校区区别
  details: 四个校区在管理、资料、考试、教学安排上是统一的，教师每学年轮换一次。不同校区的区别在于生源、管理细节、硬件条件等。
